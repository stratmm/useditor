Overview

The Log class provides a static interface to output text to STDOUT. It provides helpful messages types such as debug, error and ok. On most command line interfaces, Log will also output text from various messages types in an appropriate color.

Examples:

Sending a log message

Useditor::Log.fatal "My fatal error is", exception.class.name
