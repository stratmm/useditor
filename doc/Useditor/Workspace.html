Overview

The Workspace class handles the working storage of the image being manipulated as well as providing the image manipulation functions.

Examples:

Creating a default image default 10 x 10 image of color W

image = Useditor::Workspace.new
image.create

Creating a new empty image with an optional background color

100x100 pixels of color B
image = Useditor::Workspace.new
image.create(rows: 100, cols: 100, color: "B")

def create(rows: 10, cols: 10, color: "B")
  delete_state
  get_state
  @rows = rows
  @cols = cols
  @image = Array.new(rows) do
    Array.new(cols) { color }
  end
  save_state
  self
end
